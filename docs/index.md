## How to Contribute

1. **Clone** the repo and install dependencies (see the [README](https://github.com/YOUR-ORG/YOUR-REPO)).
2. **Edit** or create a `.md` file inside `docs/`.
3. **Push** to `main` — the site rebuilds and deploys automatically in ~1–2 minutes.

!!! tip "Quick edits without cloning"
    Every page has a :material-pencil: **edit** button (top right) that opens the file directly in GitHub's web editor.
