# Papers

All papers the group has read, is reading, or plans to read. Keep the table updated
as you work through papers. Add notes inline or link to a longer write-up in
[Findings](../findings/index.md).

**Status key:**

| Badge | Meaning |
|---|---|
| To Read | On our list but not yet started |
| Reading | Actively being read by someone |
| Read | Finished (and summarised?) |
| Referenced | Cited in our work / used as background |

## How to Add a Paper

Add a new row to the table above:

```markdown
| [Title](URL) | Authors | Year | Venue | Topic | STATUS EMOJI | Notes |
```

Where `URL` is the arXiv link, DOI, or PDF URL.

## Useful Search Resources

- **[arXiv cs.CC / cs.DS / math.CO](https://arxiv.org/)** — preprints in complexity, algorithms, combinatorics
- **[Semantic Scholar](https://www.semanticscholar.org/)** — citation search and "related papers" graph
- **[Connected Papers](https://www.connectedpapers.com/)** — visualise the citation neighbourhood of a paper
- **[ECCC](https://eccc.weizmann.ac.il/)** — Electronic Colloquium on Computational Complexity
- **[DBLP](https://dblp.org/)** — comprehensive CS bibliography
